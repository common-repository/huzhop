Support WooCommerce order fulfillment for dropshipping, and image variations color showing in the product page.

= Features =

* Fulfillment for WooCommerce Order
* Woocommerce product variations swatches, Showing thumbnail images for variations on WooCommerce product.

with this wordpress huzhop plugin, You can use suitable products with many variants in WooCommerce system.

== Installation ==

1. Upload the plugin to your /wp-content/plugins/ and activate it (alternatively, install through the WP admin console)
2. Make Sure you have latest version of WordPress(or minimum 4.8)
2. Go to Settings > Huzhop , for custom variant type if it is image
4. Go to WooCommerce > Orders, on the right side you can see Huzhop fulfillment section
5. Click "Mark as fulfilled" button if you want that order fulfilled.
